👨🏻‍🏫 Metode Newton-Raphson

Aplikasi perhitungan untuk menentukan akar persamaan linear dengan metode Newton-Raphson.

Follow on Github

💪🏻 Bantuan
👩🏻‍🏫 Syarat Penggunaan

Perhitungan dapat dilakukan ketika soal memenuhi syarat-syarat berikut.

  1. Terdapat akar persamaan linearnya
  2. Terdapat nilai awal atau tebakan awal
  3. Terdapat nilai toleransi galat relatif 𝑥
👨🏻‍💻 Tips Penginputan
  • Gunakan titik (.) untuk penggunaan koma
  • Gunakan simbol bintang (*) untuk perkalian
  • Gunakan simbol bintang 2 (**) untuk bilangan pangkat
  • Tidak dapat menggunakan variabel selain 𝑥
  • Gunakan variabel 𝑥 dengan huruf kecil bukan huruf kapital
